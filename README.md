## Prerequisites

- npm >=5.5.0
- node >=9.3.0

## Install

```bash
# Install Server side dependencies
npm install

# Install Client side dependencies
npm run client-install
```

## Usage

```bash
# install concurrently globally
npm i -g concurrently

# Run Client & Server  
npm run dev
```

## Run tests

```sh
npm run test
```
